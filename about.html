  • Event Overview



    The Code Quest Challenge is not your typical coding competition; it's an immersive and thrilling experience that combines coding skills, problem-solving, and teamwork with a dash of mystery and fun.

  • Balloon Debugging

    In this round, participants will need to debug given source code obtained by popping balloons. Here's how it works: Balloons of different colors will be placed around the room. Each color represents a specific programming language. Teams will select a balloon of their choice and pop it to retrieve the source code printed inside. Once the source code is obtained, the team must debug it to find the correct output. One member from each team will write the correct output on the provided board. Teams have a time limit of 3 minutes to debug the code and write the output on the board.

  • The EscapeRoom Challenge



    Round 2: The EscapeRoom Challenge


    Congratulations to the winners of Round 1! You've made it to the next exciting stage of the competition: Round 2 - The Laptop Challenge.

    The Challenge
    In Round 2, each winning team will be given a laptop with a locked program, and you'll be tasked with a coding challenge like no other. It's a test of your coding skills, teamwork, and the ability to think on your feet.

    The Laptop Setup
    Each team will receive a laptop that contains a coding challenge program. However, there's a twist - the program contains an incorrect code snippet with bugs that need fixing.

    The Coding Challenge
    Your mission is to identify and correct the bugs in the given code snippet to make it fully functional.You'll see a full program with incorrect code snippets. Your task is to locate and fix the issues. The challenge is to identify the incorrect elements (variable names, function names, or code segments) and replace them with the correct ones. But that's not all - read on to discover the added twist!

    Time Limit
    You'll have a total of 25 minutes to complete the coding challenge and submit your corrected program.

    Password Entry
    After fixing the code you'll be redirected to a password entry screen. You have three chances to correctly input the password based on the clue you received. For example, if the clue was "The password is the name of a famous programming language backward," the correct password would be "elgnimmargorp." But the clue will be related to opensource.
