  • Event Overview



    SourceSphere is a competition designed to engage participants in debugging code snippets and solving algorithmic puzzles. The event is divided into three rounds, each with its own unique challenge. Teams of three participants will battle it out to showcase their coding expertise.

  • Algorithm Jumble

    In this round, participants will face the challenge of arranging algorithmic code pieces that have been scrambled. The rules are as follows: A bowl will contain jumbled pieces of two different algorithms. Each team will have 2 minutes to unscramble and arrange the code pieces correctly to reconstruct the original algorithms. Teams can choose to solve as many algorithms as they can within the given time frame, but they must aim to solve a maximum of 5 algorithms.

  • Balloon Debugging

    In this round, participants will need to debug given source code obtained by popping balloons. Here's how it works: Balloons of different colors will be placed around the room. Each color represents a specific programming language. Teams will select a balloon of their choice and pop it to retrieve the source code printed inside. Once the source code is obtained, the team must debug it to find the correct output. One member from each team will write the correct output on the provided board. Teams have a time limit of 3 minutes to debug the code and write the output on the board.
